Welcome

This is the testing ground for a new RPG framework built entriely in JavaScript. The eventual goal is to integrate the code into Unity3D once the core logic has been built.

The project will follow best practices with Object Oriented Programming and Design within the JavaScript language.

Project Manefesto

  • Object Oriented JavaScript
  • Maintainable code through consitent use of patterns
  • Web based only for testing

Use Cases

Character.JS
  1. The player creates a character for the game.
Encounter.JS
  1. The game initiates an encounter for the player's character.
  2. The game checks to see if there is a surprise round.
  3. The player can chose to enter or flee the encounter.
  4. A player in an encounter controls their character's actions.
  5. The game controls enemy from available actions.
  6. A player in the encounter either wins or loses.
    • If the player wins they gain experience (XP) and treasure (GP).
    • If the player loses they have all teasure removed.
  7. The player will gain a new level after accumulating enough expereince.
