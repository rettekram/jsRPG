Character

Creates a new instance of a Character object for use in game encounters.

Character.JS

Use Case
  • The player creates a new character for the game.
    • The player rolls dice to generate the character's attribute points.
    • The player assigns attribute points to available attributes.
    • The player chooses a Class for the character.
    • The player chooses a Race for the character.
    • The player chooses X Skills for the character.
    • The game sets default Hit Point value to character.
    • The game sets character Level to 1.
  • The player views an exiting character.
  • The player edits an existing character.
